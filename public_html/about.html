About

Floored is a price tracker platform that allows users to enter the URL of a product and add the product to a repository of watched items. Users will receive email alerts when the price of a product drops. Users can also sign in to the web application to view a dashboard of the tracked price changes of a product.

This application is built using the LAMP stack with Materialize CSS as the front-end framework. The founders of this project are Sam and Angel, two students at the University of Texas at Austin. This is a final project for the CS 329E: Web Programming course.
